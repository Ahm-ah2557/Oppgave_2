Teknologi og demokrati

Teknologiske endringer har det siste tiåretfundamentalt forandret hvordan vi mottar
og bruker informasjon, og flere land har sett store politiske endringer i kjølvannet av
denne teknologiske utviklingen. Teknologi har både positive og negative sider.

Fordeler med teknologi!

Det er så mange fordeler med teknologi i vårt daglige liv. Med hjelp av mobilteknologi kan vi snakke med våre venner og slektninger som bor langt fra oss. Med hjelp av Internett kan vi lære nye ting og online kurs. vi kan dele informasjon til alle hele verden i løpet av øyeblikk. Økt effektivitet og produktivitet: Teknologi kan hjelpe oss med å utføre oppgaver raskere og mer effektivt. Bedre kommunikasjon: Teknologi som telefoner, e-post og sosiale medier gir oss muligheten til å kommunisere med hverandre fra hvor som helst i verden

Ulemper med teknologi!

Avhengighet: Folk kan bli avhengige av teknologi og bruke det for mye, noe som kan påvirke sosial interaksjon, søvn og helse. folk bruker stadig mer tid på å spille videospill, lærer å bruke moderne teknologi, bruker sosiale nettverk og surfer på Internett, ignorerer deres virkelige liv. Jobb: Teknologi kan erstatte arbeidsplasser og føre til jobb, spesielt i industrier som er automatisert.
